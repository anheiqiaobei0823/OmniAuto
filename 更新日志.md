# Changelog

## v0.1.1 (2026-07-08)

### 新增

- **管理员面板**：新增独立的管理员标签页，包含用户管理（创建/删除/重置密码）和注册开关控制
- **移动端适配**：导航菜单支持移动端折叠展开；图表在移动端自动切换为竖排显示
- **路由模型导入**：设置页新增路由模型导入功能
- **模型入口排序箭头**：每个模型右侧增加 ▲▼ 排序按钮
- **Hash 路由**：URL 支持 `#dashboard`/`#模型入口` 等锚点，浏览器前进后退可用
- **分类添加弹窗**：添加分类改用弹窗输入，替代浏览器原生 prompt
- **用户密码重置**：管理员可为任意用户重置密码

### 修复

- 修复 API Key 禁用功能失效的问题（已禁用的 Key 仍可调用）
- 修复非管理员用户登录后主页空白的问题
- 修复非管理员用户无法操作供应商/模型/Key/路由的问题
- 修复管理员验证接口崩溃导致 POST/PUT 返回 500 的问题
- 修复本地开发时 `.env` 配置文件无法读取的问题

### 改进

- 清除数据：改为每人只清除自己的数据，不再误删全局
- URL 前缀：管理后台地址从 `/admin` 改为 `/OmniAuto`
- 标签重命名：「用量」→「用量明细」，「记录」→「使用明细」，「路由」→「模型入口」
- 操作按钮布局：供应商和 Key 页面的操作按钮在移动端整组换行
- 模型入口交互：移除拖拽排序，改用箭头按钮；图标尺寸统一放大
- 移除主页和记录页的冗余刷新按钮

## v0.1.0 (2026-07-08)

### 新增

- 项目正式更名为 OmniAuto
- 用户认证系统：JWT 登录、注册开关、用户管理、修改密码
- 管理后台所有接口接入 JWT 鉴权
- 首页 API 地址一键复制
- API Key 支持显示/隐藏与一键复制
- 用量统计支持日期范围筛选（昨天/今天/近3天/近7天）
- 供应商支持 Kelivo 格式批量导入
- API Key 权限选择按供应商分组折叠
- 路由模型选择按供应商分组

### 改进

- 模型颜色分配改为按顺序分配，避免相近色撞车
- Token 显示改为原始数字 + 千分位（如 1,000,000 tokens）
- 心跳检测间隔调整为 30 分钟（GET /models，不消耗 token）
- 默认分类支持添加/移除模型

### 修复

- 修复管理后台密码修改不生效的问题
- 修复调用记录时间显示错误（时区不一致）
- 修复流式请求 Token 统计为 0 的问题
- 修复使用明细中 Key 名称不显示的问题
- 修复 `/v1/models` 返回所有模型的问题，现在只暴露 `auto`
