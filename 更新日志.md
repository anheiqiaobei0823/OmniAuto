# Changelog

## v0.1.1 (2026-07-08)

### 新增

- **用户隔离**：每个用户的供应商、API Key、路由分类完全独立，互不可见
- **管理员标签页**：独立「管理员」Tab（含用户管理、注册开关），仅管理员可见
- **手机端适配**：汉堡菜单全局共享，所有页面可切换；图表移动端竖排显示
- **路由排序箭头**：每个模型右侧 ▲▼ 按钮，手机端可轻松调序
- **Hash 路由**：URL 支持 `#dashboard`/`#routing` 等，浏览器前进后退可用
- **一键导入供应商**：设置页新增导入按钮，直接粘贴字符串批量导入
- **用户密码重置**：管理员可为任意用户重置密码（弹窗操作）
- **分类添加弹窗**：添加分类改为弹窗输入，取代浏览器 prompt

### 修复

- **verify_admin 崩溃**：返回正确的 Request 对象，POST/PUT 不再 500
- **API Key 禁用不生效**：dislabed 的 Key 现在正确返回 401
- **非管理员主页空白**：普通用户正常显示仪表盘和调用地址
- **非管理员操作权限**：所有用户可操作供应商/模型/Key/路由
- **模型名**：`/v1/models` 返回 `OmniAuto`（原 `auto`）
- **模型能力标签**：返回完整 `capabilities`（vision/tools/reasoning/stream/image_gen）
- **refreshDashboard 崩溃**：非管理员进入主页不再 403
- **加载 dotenv**：本地开发可正常读取 `.env` 配置

### 改进

- 清除数据：改为每人清自己的数据，不再误删全局
- 导入文本提示：改为"粘贴其他模型上导出的字符串或 JSON"
- 删除刷新按钮：主页和记录页冗余按钮已移除
- URL 前缀：管理后台改为 `/OmniAuto`（原 `/admin`）
- Docker：新增 `.dockerignore`，排除本地文件
- 标签重命名：「用量」→「用量明细」，「记录」→「使用明细」，「路由」→「模型入口」
- 操作按钮整组换行：供应商和 Key 页面按钮移动端不散排
- 移除拖拽排序：模型入口用箭头替代拖拽，图标放大统一高度
- 设置页权限：GET 开放给所有用户，PUT 仍管理员专属
