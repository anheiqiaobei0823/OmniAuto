# Changelog

## v0.1.0 (2026-07-08)

### New Features
- 项目正式更名为 OmniAuto
- 用户认证系统：支持 JWT 登录、注册开关、用户管理、修改密码
- 管理后台所有接口接入 JWT 鉴权
- 首页 API 地址一键复制
- API Key 支持显示/隐藏与一键复制
- 用量统计支持日期范围筛选（昨天/今天/近3天/近7天）
- 供应商支持 Kelivo 格式批量导入
- API Key 权限选择按供应商分组折叠
- 路由模型选择按供应商分组

### Improvements
- 模型颜色分配改为按顺序分配，避免相近色撞车
- Token 显示改为原始数字 + 千分位（如 1,000,000 tokens）
- 心跳检测间隔调整为 30 分钟（GET /models，不消耗 token）
- 默认分类支持添加/移除模型

### Bug Fixes
- 修复管理后台密码修改不生效的问题
- 修复调用记录时间显示错误（时区不一致）
- 修复流式请求 Token 统计为 0 的问题
- 修复使用明细中 Key 名称不显示的问题
- 修复 `/v1/models` 返回所有模型的问题，现在只暴露 `auto`
